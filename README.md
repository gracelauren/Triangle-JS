Triangle-JS
==============

by Grace Mekarski

Triangle-JS is a web page that lets the user input the lengths of the sides of a triangle, and returns whether they form an equilateral, isosceles, or scalene triangle.

Installation
------------

Install Triangle-JS by first cloning the repository.  
```
$ git clone https://github.com/gracelauren/Triangle-JS.git
```

Open Triangle-JS by going into the folder and clicking index.html or in the terminal type:
```
$ cd Triangle-JS
```
```
$ open index.html
```

Test it out!
```
